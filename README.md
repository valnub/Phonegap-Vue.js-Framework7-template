## Compatibility

- Framework7 v1
- Vue 2.x or higher

## Usage

Make sure you have Phonegap/Cordova installed:

```shell
npm install -g phonegap@latest
```

To start a new build and run the project, go to your project folder and do:

```shell
cd www
npm run build
cd ..
phonegap run ios
```
